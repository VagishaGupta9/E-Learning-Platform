Lisp (List Processing):

Lisp is a programming language known for its unique approach to symbolic computing and list processing. Here are some salient features and syntax:

  • Features:
    • Dynamic typing: Variables can hold values of any data type.
    • Garbage collection: Automatic memory management.
    • Powerful macro system: Allows for code manipulation and abstraction.
    • Interactive development: Supports an interactive REPL (Read-Eval-Print Loop).
  • Syntax:
    • Lists: Fundamental data structure; operations are expressed as function calls within parentheses.
    • Example function call: (+ 1 2) results in 3.
    • Defining a function: (defun add-two-numbers (a b) (+ a b)).

Prolog:

Prolog is a logic programming language designed for symbolic reasoning and manipulation. Here are some salient features and syntax:

  • Features:
    • Declarative programming: Programs express what should be achieved rather than how to achieve it.
    • Pattern matching: Powerful unification mechanism for matching facts and rules.
    • Backtracking: Allows the system to explore multiple solutions.
    • Extensibility: Easily extend the language through custom predicates and rules.
  • Syntax:
    • Facts: father(john, jack) asserts that John is the father of Jack.
    • Rules: ancestor(X, Y) :- parent(X, Y) defines a rule for ancestry.
    • Queries: ?- ancestor(john, jack) queries if John is an ancestor of Jack.
